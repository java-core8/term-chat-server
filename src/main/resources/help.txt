Программа @Chat
Отправка сообщений по нажатию <Enter>

Выполнение команды через "/" перед именем команды. Пример: /help
Параметры команды передаются через разделитель "=". Пример: /command=param
Если последовательно указано несколько команд, распознается только первая.

Список команд:
command[=param] [message] {direction<private | all>}

help            {private}    -   выводит справку пользователя.
to=user message {private}    -   отправляет [message] для [user]. Положение команды в тексте не важно.
history=count   {private}    -   присылает лог чата в количестве [count] сообщений. Ограничение 50 сообщений.
list            {private}    -   присылает список пользователей в чате
exit [message]  {all}        -   разрывает соединение с сервером. message опционально.
